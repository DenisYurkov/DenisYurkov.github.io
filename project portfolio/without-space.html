Without Space

Video Gameplay:


About the project:

Game Engine: Unity

Category: Solo Game

Published: itch.io

Game Was Made: 22.11.2020

Control:

W/S/D/A - Move,

E - Open postbox,

F - Play Cassette.


Description of the game:

You were gone for a long time, after the death of your grandmother, you find the cassettes that you recorded for yourself a long time ago, you find out that your grandmother left you a farewell note, and you must find it.

Interesting facts about the game: For the game, I needed voice acting, I asked a British friend to help me with voice acting, my friend Arthur also wrote a very good soundtrack for the game, which perfectly suits the mood of the game, it was an unforgettable experience.

Code:

Unfortunately, the code was not saved, I can only say in words what I was doing in the game, I created the character control using the Сharacter Controller, worked with OnTriggerEnter to move between locations, with OnTriggerStay for cassettes, changed the sound directly inside the code, got acquainted with the excellent Audacity program for sound editing, I got acquainted with how to create UI in Unity, how to bake the light correctly, how to change the skybox, since this was my first game, I didn't really think about optimization, so the game may slow down at some points, but I'm learning from my mistakes, and in my new projects I try to avoid it.
