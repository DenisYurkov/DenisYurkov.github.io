About the project:

Game Engine: Unity

Category: Solo Game

Control:

Left Mouse Button - Interact,

W/S/D/A - Move,

Left Shift - Run,

Space - Jump,

Mouse - Look Around,

F - Flashlight,


Description of the game:

Behind The Wind - it is the atmosphere of Egyptian tombs, adventure, riddles, and running along the walls.
The game was made 22.06.2021, for the Level Design module at my university, I made the game for 3 weeks.
